# Robo Speaker 🗣️

**Robo Speaker** is a simple Python-based text-to-speech (TTS) application that converts typed text into speech using the `pyttsx3` library. It's lightweight, runs offline, and works across platforms — ideal for beginners exploring speech synthesis with Python.

---

## 📌 Features

- 🔊 Converts text to speech
- 💻 Cross-platform (Windows, macOS, Linux)
- ⚡ Offline and lightweight
- 🧠 Beginner-friendly Python script

---

## 🛠️ Built With

- Python 3
- [`pyttsx3`](https://pypi.org/project/pyttsx3/) - Offline text-to-speech library
